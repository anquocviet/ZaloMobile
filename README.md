## Install environment and reference source before build and run

[Expo: Local app development](https://docs.expo.dev/guides/local-app-development/)

## Build

npx expo prebuild

### ios

npx expo run:ios

### android

npx expo run:android

## Run

### ios

**npm:** npm run ios

**yarn:** yarn ios

### android

**npm:** npm run android

**yarn:** yarn android
